offset value

Return Value

CBUUID generated from base and addOffset

Discussion

Generate CBUUID given base of type yms_u128_t and offset of type int

This method is used in place of [YMSCBUtils genCBUUID:withIntOffset:] due to the peculiar way (IMHO) the TI SensorTag 128-bit address map is setup: Offset values are put into the 96th bit position of a 128-bit address.
