YMSCBPeripheral.m

Overview

YMS CoreBluetooth representation of a peripheral.

This is a container class which manages an instance of CBPeripheral and its associated delegate callbacks.

Note that the CBPeripheralDelegate method peripheral:didUpdateValueForCharacteristic:error: will call [YMSCBService updateCharacteristic:] to handle responses to read and writes made to that service.
